# 2026-05-14

## PyTorch 2.12 RSV 胶囊诊断与修复
- 诊断问题：用户反馈 PyTorch 2.12 升级后 RSV 胶囊工作流结束时显示异常
- 分析工具：创建 `diagnose_rsv.py` 诊断脚本
- 诊断结果：
  - `memory_reserved()` 本身工作正常
  - PyTorch 2.12 XPU 缓存分配器行为变更：张量释放后缓存池立即归还驱动，`reserved` 归零
  - RSV=0 时胶囊用 `xpusys-na` (灰色 #555) 导致看起来像"没生效"
- 修复：`renderRSV()` 中 RSV=0 时不再使用 `xpusys-na`，改用默认主题颜色（空字符串）
- 诊断脚本 `diagnose_rsv.py` 保留在项目目录中

## GPU 规格表 + 新胶囊设计讨论
- 阅读 blackwood.cv GPU AI Perf Assembly 博客文章，讨论在 ComfyUI 工作台中新增 GPU 规格信息展示

- 明确 GGUF Q4_K_M 的存储精度=4-bit，计算精度=FP16/BF16

## PCI ID 数据收集
- NVIDIA：Pascal~Ada Lovelace PCI ID 从 pci.ids 官方数据库确认（36 张卡）
- AMD：RDNA 1~4 PCI ID 从 DeviceHunt 确认（10 张卡，注意多型号共用芯片 ID）
- Intel：Arc 全系 PCI ID 从现有 ARC_PCI_TGP 复用并补充（20 张卡）
- 缺口：NVIDIA Blackwell RTX 50 系列桌面版 PCI ID 未公开；Intel Arc A310 / Pro B65 / Pro B70 PCI ID 未找到
